## Overview

A systematic checklist for verifying the accuracy of manuscript content. Use this checklist for each chapter before it moves to the final editing stage.

## Per-Chapter Checklist

- [ ] All dates verified against primary or authoritative secondary sources
- [ ] All statistics and data points verified against original datasets
- [ ] All direct quotations checked against transcripts or original texts
- [ ] All attributions confirmed (correct person, correct context)
- [ ] All proper names spelled correctly and consistently
- [ ] All titles of works, organizations, and positions verified
- [ ] All locations and geographic details verified
- [ ] All historical claims cross-referenced with at least two sources
- [ ] All scientific or technical claims reviewed by subject-matter expert
- [ ] All URLs and web references checked for accessibility
- [ ] All citations match entries in [[Master-Bibliography]]
- [ ] All permissions secured for quoted material (see [[Permissions-Tracker]])

## Common Pitfalls

- Dates that shift between sources (especially for events before modern record-keeping)
- Statistics that have been updated since the source was published
- Quotes that have been slightly altered in secondary sources
- Names with multiple transliterations or variant spellings
- Claims that "everyone knows" but lack a specific source
